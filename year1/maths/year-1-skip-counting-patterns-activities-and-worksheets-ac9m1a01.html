About This Year 1 Maths Topic

Skip counting helps students recognise number patterns and build early algebraic thinking. Instead of counting every number, students count forwards in equal jumps such as 2, 4, 6, 8 or 5, 10, 15, 20.

In this unit, students use number charts, songs, rhymes, stories, blocks, beads, coins, notes and movement games to explore skip-counting sequences by 2s, 5s and 10s.

Students learn to recognise the rule in a sequence, continue a pattern, identify missing numbers and create their own number, shape and object patterns.

They also apply skip counting to practical situations such as counting equal-value coins and notes, packing objects into groups and playing counting games.

These activities support the Australian Curriculum Version 9.0 content descriptor AC9M1A01 and its associated elaborations.

Learning Intentions

Students will learn to:

  • recognise skip-counting patterns;
  • count forwards by 2s, 5s and 10s;
  • continue number sequences using a pattern rule;
  • identify missing numbers in skip-counting sequences;
  • create skip-counting sequences with numbers;
  • represent growing patterns using shapes and objects;
  • describe what changes and what stays the same in a pattern;
  • recognise that counting by 5s produces numbers ending in 0 or 5;
  • use skip counting to count equal-value money;
  • use games, songs and movement to practise number patterns.

Teacher Guide

Recommended Materials

  • hundreds charts and number tracks;
  • number cards from 0 to 120;
  • counters, cubes, blocks or bottle caps;
  • beads and string;
  • shape cards and pattern strips;
  • 5-cent, 10-cent and $2 play coins;
  • $5 and $10 play notes;
  • small boxes, packets and classroom objects;
  • robot role-play cards or simple props;
  • mini whiteboards and markers;
  • songs, rhymes and counting stories;
  • floor number lines or chalk number tracks.

Suggested Teaching Sequence

  1. Review counting forwards by ones.
  2. Introduce equal jumps on a number line.
  3. Practise counting by 2s with movement and songs.
  4. Practise counting by 5s using hands, clocks and money.
  5. Practise counting by 10s using bundles and number charts.
  6. Highlight skip-counting numbers on a hundreds chart.
  7. Build growing patterns with blocks, beads and shapes.
  8. Identify and describe the rule in each sequence.
  9. Apply skip counting to equal-value coins and notes.
  10. Consolidate learning with robot role-play and counting games.

Teacher Questions

  • What number comes next?
  • How much are we adding each time?
  • What is the pattern rule?
  • Which numbers did we skip?
  • How can the number chart help us?
  • What changes in this growing pattern?
  • How many objects are added each time?
  • What do you notice about numbers counted by 5s?
  • Why do numbers counted by 10s end in zero?
  • How can skip counting help us count these coins?
  • Can you show the same pattern with objects?
  • Can you create another sequence using the same rule?

Common Misconceptions

Counting by ones instead of equal jumps

Use a number line and physically jump over the numbers that are not said. Emphasise that each jump must be the same size.

Changing the skip-counting rule part-way through

Ask students to state the rule before continuing the sequence and check the difference between neighbouring numbers.

Confusing counting by 5s with adding 5 once

Show repeated jumps of 5 on a number line and connect each jump to one more equal group of five.

Counting the number of coins instead of their value

Remind students that each coin has the same value. Label each jump with the coin value and count the total amount.

Focusing only on the colour of an object pattern

Ask students to describe the number of objects in each stage and identify how many are added each time.

Thinking a pattern must always start at zero

Show skip-counting patterns that begin at different starting numbers, such as 3, 5, 7, 9 or 6, 11, 16, 21.

Year 1 Skip Counting Patterns Activities

Activity 1: Hundreds Chart Hop

Students highlight or cover numbers while counting by 2s, 5s and 10s on a hundreds chart.

Ask them to describe the visual pattern formed in each column or row.

Curriculum connection: AC9M1A01_E1

Activity 2: Skip-Counting Song and Actions

Sing a counting song while students clap, stamp or jump on each number in the sequence.

Change the action for counting by 2s, 5s and 10s.

Curriculum connection: AC9M1A01_E1

Activity 3: Build a Growing Block Pattern

Build towers containing 2, 4, 6, 8 and 10 blocks. Place the towers in order and identify how many blocks are added each time.

Repeat the activity using groups of 5.

Curriculum connection: AC9M1A01_E2

Activity 4: Bead String Patterns

Students create bead groups that grow by 2 or 5 and record the matching number sequence.

Ask them to predict the number of beads in the next group.

Curriculum connection: AC9M1A01_E2

Activity 5: What Do You Notice?

Display 0, 5, 10, 15, 20, 25 and 30. Students circle the final digit and describe the repeating pattern.

Compare this sequence with counting by 10s.

Curriculum connection: AC9M1A01_E3

Activity 6: Missing Number Detective

Give students skip-counting strips with missing numbers, such as 10, 20, __, 40, __.

Students explain the rule and fill in the missing values.

Curriculum connection: AC9M1A01_E3

Activity 7: Count the Money

Give students collections of equal-value play coins or notes. Students count the total using 2s, 5s or 10s.

Ask them to record both the skip-counting sequence and the total value.

Curriculum connection: AC9M1A01_E4

Activity 8: Robot Packing Line

One student acts as a robot and packs 5 or 10 objects into each box. The class keeps a running total.

Students predict the total after the next box is packed.

Curriculum connection: AC9M1A01_E5

Activity 9: Sam-yuk-gu Counting Game

Students count around a circle and replace selected numbers with a clap, action or word.

Adapt the game to focus on multiples of 2, 5 or 10.

Curriculum connection: AC9M1A01_E6

Activity 10: Create Your Own Pattern

Students choose a starting number and a skip-counting rule, then create a number, shape or object sequence.

A partner identifies the rule and continues the pattern.

Curriculum connection: AC9M1A01

Differentiation

Support

  • begin with counting by 10s before 5s and 2s;
  • use physical objects for every skip-counting sequence;
  • provide number charts with target numbers highlighted;
  • use short sequences with only one missing number;
  • start skip counting from zero;
  • use repeated songs and movement routines;
  • provide pattern rule cards such as “add 2”;
  • pair students for oral counting practice;
  • allow students to point or move counters while counting.

Extension

  • begin sequences from numbers other than zero;
  • count backwards by 2s, 5s and 10s;
  • create missing-number puzzles for classmates;
  • compare two different skip-counting sequences;
  • continue patterns beyond 100;
  • describe the position of numbers on a hundreds chart;
  • use mixed coin collections with more than one denomination;
  • write a story problem involving equal groups;
  • explain why a pattern rule always works.

Assessment Ideas

Observe whether the student can:

  • count forwards by 2s, 5s and 10s;
  • continue a skip-counting sequence;
  • identify missing numbers in a pattern;
  • state the rule for a sequence;
  • represent a growing pattern with objects;
  • recognise patterns in the final digits of numbers;
  • count equal-value coins or notes efficiently;
  • keep a running total in a grouping activity;
  • participate in a skip-counting game;
  • create and explain an original pattern.

Quick Exit Ticket

  1. Continue: 5, 10, 15, _____, _____.
  2. State the pattern rule.
  3. Show the pattern using counters.
  4. Write one new sequence using the same rule.

Home Learning Ideas

Families can practise skip counting using socks, pegs, blocks, coins, toy animals, pasta pieces or other safe household items.

Arrange the objects into equal groups of 2, 5 or 10 and count the running total aloud.

Ask questions such as:

  • What number comes next?
  • How much are we adding each time?
  • What is the pattern rule?
  • Can you count backwards using the same rule?
  • Can you show the pattern with objects?
  • What numbers did you skip?
  • What do you notice about the final digits?
  • How many groups are there?
  • What is the total value of the coins?
  • Can you create a new skip-counting pattern?

About This Year 1 Maths Topic

Data displays help students organise information so that it can be counted, compared and discussed clearly.

In this unit, students represent categorical data using objects, drawings, pictographs, tally charts and digital tools.

They learn that one object or drawing can represent one data value and use counting and skip-counting strategies to determine the frequency of each category.

Students compare categories, identify the most and least popular responses, discuss empty categories and explain what the data display shows.

These activities support the Australian Curriculum Version 9.0 content descriptor AC9M1ST02 and its associated elaborations.

Australian Curriculum Alignment

AC9M1ST02

Represent collected data for a categorical variable using one-to-one displays and digital tools where appropriate; compare the data using frequencies and discuss the findings.

AC9M1ST02_E1

Create a pictograph using objects or drawings, discuss suitable categories, arrange the data into those categories and reflect on whether the chosen categories were helpful.

AC9M1ST02_E2

AC9M1ST02_E3

AC9M1ST02_E4

Represent data using objects and drawings where one object or drawing represents one data value, and explain patterns using counting strategies.

AC9M1ST02_E5

Discuss frequency as the total count for a category and relate the highest frequency to the most popular category.

AC9M1ST02_E6

Explore First Nations Australian children’s instructive games, including Kolap from Mer Island in the Torres Strait region, and record, represent and discuss the outcomes.

Learning Intentions

Students will learn to:

  • represent categorical data in a pictograph;
  • select useful categories for a data display;
  • use one object or drawing for one data value;
  • translate a list or picture display into a tally chart;
  • count tally marks efficiently using groups of five;
  • identify and represent zero-frequency categories;
  • compare category frequencies;
  • identify the most and least popular categories;
  • describe patterns in a data display;
  • discuss findings using clear statistical language.

Teacher Guide

Recommended Materials

  • sorting objects and category labels;
  • picture cards and drawings;
  • pictograph templates;
  • tally chart templates;
  • lists of collected responses;
  • stickers or reusable symbols;
  • mini whiteboards and markers;
  • digital chart or pictograph tools;
  • frequency cards;
  • zero and blank category examples;
  • counting frames or counters;
  • authoritative First Nations game resources.

Suggested Teaching Sequence

  1. Review categories and collected data.
  2. Introduce one-to-one representation.
  3. Create pictographs with objects or drawings.
  4. Discuss whether the chosen categories are useful.
  5. Translate pictograph or list data into tally charts.
  6. Use groups of five to count tallies efficiently.
  7. Explore zero-frequency and empty categories.
  8. Compare frequencies across categories.
  9. Identify the most and least popular categories.
  10. Discuss findings and reflect on the display.

Teacher Questions

  • What does each object or drawing represent?
  • Are the categories clear and useful?
  • Could any response fit in more than one category?
  • How can we change this list into a tally chart?
  • How many groups of five can you see?
  • What is the frequency of this category?
  • Which category has the highest frequency?
  • Which category has the lowest frequency?
  • What should we do with a category that has no data?
  • What pattern do you notice?
  • How does this display answer the question?
  • Could the display be improved?

Common Misconceptions

Using one picture to represent different amounts

Reinforce that in a one-to-one display, each object or drawing represents exactly one data value.

Uneven spacing that makes categories look larger

Align objects or drawings consistently so visual length does not mislead students about frequency.

Miscounting tally marks

Encourage students to identify complete groups of five before counting remaining marks.

Thinking a blank category means missing work

Explain that a blank may deliberately represent a frequency of zero when no responses belong to that category.

Confusing most popular with the first category

Compare frequencies rather than category position or label order.

Choosing overlapping categories

Review the category labels and adjust them so each response has one clear place where possible.

Year 1 Data Representation and Pictograph Activities

Activity 1: Build an Object Pictograph

Give students collected data and real objects or counters to arrange into categories.

Discuss whether the category labels help answer the original question.

Curriculum connection: AC9M1ST02_E1

Activity 2: Draw a Class Pictograph

Represent class preference data using one drawing for each response.

Students compare the categories and describe what the graph shows.

Curriculum connection: AC9M1ST02_E1

Activity 3: List to Tally Chart

Provide a list of responses and ask students to translate it into a tally chart.

Compare the list and tally chart to decide which is easier to count.

Curriculum connection: AC9M1ST02_E2

Activity 4: Count Tallies by Fives

Students identify complete tally groups of five and use skip counting to determine each category total.

They explain how the chart answers the data question.

Curriculum connection: AC9M1ST02_E2

Activity 5: Empty Birthday Month

Create a class birthday display and identify any month with no birthdays.

Discuss whether to show zero, leave a blank or omit the category.

Curriculum connection: AC9M1ST02_E3

Activity 6: Zero or Blank?

Show several data displays with empty categories and ask students which presentation is clearest for each purpose.

Students explain their choice.

Curriculum connection: AC9M1ST02_E3

Activity 7: One Object, One Vote

Students place one counter or drawing in a category to show one response.

Count the objects and describe patterns in the completed display.

Curriculum connection: AC9M1ST02_E4

Activity 8: Most and Least Popular

Compare category frequencies in a pictograph or tally chart.

Students identify the most popular, least popular and equal categories.

Curriculum connection: AC9M1ST02_E5

Activity 9: Digital Pictograph

Use a simple digital tool to create a one-to-one pictograph from collected class data.

Compare the digital display with a paper version.

Curriculum connection: AC9M1ST02

Activity 10: Kolap Data Investigation

Explore an authoritative description of Kolap from Mer Island in the Torres Strait region.

Record outcomes, represent the results and discuss category frequencies respectfully and accurately.

Curriculum connection: AC9M1ST02_E6

Differentiation

Support

  • begin with two categories;
  • use real objects before drawings;
  • provide pre-labelled pictograph columns;
  • use small data sets;
  • model one-to-one placement carefully;
  • provide pre-drawn tally groups;
  • use visual most and least cards;
  • highlight zero-frequency categories;
  • allow oral explanations or adult scribing.

Extension

  • use more categories and larger data sets;
  • compare two different data displays;
  • create a pictograph from independently collected data;
  • translate between lists, pictographs and tally charts;
  • explain why a category should be blank, zero or omitted;
  • identify equal frequencies;
  • create questions that the display can answer;
  • evaluate whether the categories are useful;
  • design a digital data display.

  • choose and use suitable categories;
  • translate list or pictorial data into a tally chart;
  • count tally marks using groups of five;
  • identify zero-frequency categories;
  • represent one data value with one object or drawing;
  • determine category frequency;
  • identify the highest and lowest frequencies;
  • describe patterns in the data;
  • discuss findings clearly.

Quick Exit Ticket

  2. Use one picture for each response.
  3. Identify the category with the highest frequency.
  4. Explain one thing the display shows.

Home Learning Ideas

Families can collect simple categorical data about favourite foods, household objects, pets, colours or daily activities.

Represent the results using drawings, counters, tally marks or a simple digital pictograph and compare the category frequencies.

Ask questions such as:

  • What does each picture represent?
  • Which category has the highest frequency?
  • Which category has the lowest frequency?
  • Are any categories equal?
  • Is there a category with zero responses?
  • Would a blank or zero be clearer?
  • Can you turn this list into a tally chart?
  • Can you count the tallies by fives?
  • What pattern do you notice?
  • What does the display tell us?

About This Year 1 Maths Topic

Data collection helps students answer questions about people, objects, events and the world around them.

In this unit, students collect categorical data using practical methods such as sorting objects, making lists, drawing pictures, recording tally marks and placing stickers or emojis on charts.

Students discuss which method will best answer a question, review the information collected and consider how they could improve their data collection next time.

They also use simple digital tools and role-play situations, including chatbot-style yes/no surveys, to gather and represent information.

These activities support the Australian Curriculum Version 9.0 content descriptor AC9M1ST01 and its associated elaborations.

Learning Intentions

Students will learn to:

  • identify a question that can be answered with data;
  • choose a suitable data collection method;
  • collect categorical data using objects, images and drawings;
  • record information using lists;
  • record observations using tally marks;
  • organise data into categories;
  • represent data using symbols, stickers and emojis;
  • collect simple yes/no data;
  • review whether a data collection method was effective;
  • communicate data clearly to others.

Teacher Guide

Recommended Materials

  • sorting objects and category labels;
  • clipboards and recording sheets;
  • picture cards and photographs;
  • list templates;
  • tally chart templates;
  • stickers and star charts;
  • emoji cards;
  • yes/no survey cards;
  • mini whiteboards and markers;
  • tablets or simple digital chart tools;
  • virtual manipulatives;
  • authoritative First Nations stories and symbol resources.

Suggested Teaching Sequence

  1. Introduce data as information collected to answer a question.
  2. Identify simple categorical questions.
  3. Discuss possible categories before collecting data.
  4. Collect data using objects, images or drawings.
  5. Record information using lists.
  6. Introduce tally marks and tally charts.
  7. Represent data using stickers, stars and emojis.
  8. Conduct a simple yes/no chatbot-style survey.
  9. Review whether the data collection method worked well.
  10. Share and communicate what the data shows.

Teacher Questions

  • What question are we trying to answer?
  • What categories will we need?
  • How could we collect the information?
  • Which method would be easiest to use?
  • How will we record each response?
  • What does each tally mark represent?
  • How many responses are in this category?
  • What could this data help us find out?
  • Did our collection method work well?
  • What could we change next time?
  • How can symbols help communicate information?
  • What does the completed chart show?

Common Misconceptions

Collecting data without a clear question

Begin every investigation by stating the question and checking that the categories will help answer it.

Changing categories during data collection

Agree on the categories before observations begin and add an “other” category where appropriate.

Recording one item more than once

Mark or move each item after it has been recorded so every observation is counted only once.

Forming tally marks incorrectly

Model groups of five clearly: four vertical lines with the fifth crossing the group.

Using different symbols for the same category

Create a clear key so each sticker, emoji or symbol has one consistent meaning.

Assuming yes/no questions answer every investigation

Compare yes/no questions with questions that need several categories and discuss which format is more useful.

Year 1 Data Collection and Representation Activities

Activity 1: Playground Rubbish Investigation

Discuss how the class could collect information about types of rubbish found in the playground.

Try one method, review the results and suggest how the collection process could be improved.

Curriculum connection: AC9M1ST01_E1

Activity 2: Choose the Best Collection Method

Present several questions and ask students whether objects, drawings, lists, tallies or digital tools would be most suitable.

Students justify their chosen method.

Curriculum connection: AC9M1ST01_E1

Activity 3: Class Interest List

Collect information about a topic of interest, such as favourite sports teams, pets or eye colours, using a list.

Examine the list and generate questions the data could answer.

Curriculum connection: AC9M1ST01_E2

Activity 4: Rearrange the Data

Give students a mixed list of responses and ask them to reorganise the information into useful categories.

Discuss how the new arrangement makes questions easier to answer.

Curriculum connection: AC9M1ST01_E2

Activity 5: Bike Shed Tally

Decide categories before observing a school location, such as the year levels of students using the bike shed.

Record each observation with tally marks and review whether the categories were effective.

Curriculum connection: AC9M1ST01_E3

Activity 6: Tally Mark Challenge

Students practise recording classroom events, object types or picture-card categories using groups of five tally marks.

Compare tally totals with the original objects or pictures.

Curriculum connection: AC9M1ST01_E3

Activity 7: Emoji Feelings Chart

Students place an emoji on a class chart to represent how they are feeling that day.

Discuss what each emoji represents and how the class data changes over time.

Curriculum connection: AC9M1ST01_E4

Activity 8: Sticker Activity Calendar

Use stickers or symbols to represent activities on a class calendar.

Students count and compare how often different activities occur.

Curriculum connection: AC9M1ST01_E4

Activity 9: Chatbot Yes/No Survey

One student role-plays a chatbot asking a simple yes/no question while classmates respond.

Represent the answers with stickers, emojis or virtual manipulatives.

Curriculum connection: AC9M1ST01_E5

Activity 10: First Nations Data Stories and Symbols

Explore authoritative examples of stories and symbols used by First Nations Australians to represent, share and communicate information.

Discuss how symbols can organise meaning and help information be remembered and shared.

Curriculum connection: AC9M1ST01_E6

Differentiation

Support

  • begin with two clear categories;
  • use real objects before abstract symbols;
  • provide picture-supported category labels;
  • model one response at a time;
  • use pre-drawn tally columns;
  • limit surveys to simple yes/no questions;
  • provide a key for stickers and emojis;
  • allow students to point or place objects instead of writing;
  • use adult scribing for explanations.

Extension

  • use more than four categories;
  • compare two different collection methods;
  • design an original survey question;
  • identify missing or unclear categories;
  • reorganise the same data in a different way;
  • collect data using a digital tool;
  • write questions that the data can answer;
  • evaluate the fairness of a survey;
  • explain how the collection method affects the results.

Assessment Ideas

Observe whether the student can:

  • identify a data question;
  • choose useful categories;
  • select a suitable collection method;
  • record information using a list;
  • use tally marks accurately;
  • represent data using symbols, stickers or emojis;
  • collect simple yes/no responses;
  • organise categorical data;
  • review the effectiveness of a method;
  • communicate what the data shows.

Quick Exit Ticket

  1. Choose a simple class question.
  2. List two or more response categories.
  3. Collect five responses using tally marks.
  4. Explain one thing the data shows.

Home Learning Ideas

Families can collect simple data about favourite foods, pets, colours, daily weather, household objects or family activities.

Record the information with lists, tally marks, drawings, stickers or emojis and discuss which method was easiest to use.

Ask questions such as:

  • What question are we answering?
  • What categories will we need?
  • How should we record each answer?
  • Which category has the most?
  • Which category has the least?
  • Could one response fit more than one group?
  • What does each symbol mean?
  • Would a tally chart help?
  • What could we change next time?
  • What does the completed data show?
